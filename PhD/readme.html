Her er en oversikt over metoder man kan bruke for å analysere dataen. Python 3 er hovedspråket som er brukt for analysere dataen. Videre er modulene Pandas, Seaborn, Matplotlib benyttet. En Objekt-Orientert tilnærming er også benyttet, noen ganger med statiske metoder. Jeg har laget 3 hovedmoduler for å analysere dataen

Modulen analysetrening

Modulen 'analysetrening' består av en klasse som heter 'Trening'. Denne innholder en rekke statiske metoder (@staticmethods) som rydder og analyserer data

Det kan være hensiktsmessig å gruppere disse statiske metodene i en .pipe i main-filen, f.eks:

df = df.pipe(Trening.dnfCountandReplace, dnfF, LagreCsv=True)\
.pipe(Trening.changeDataType)\
.pipe(Trening.renameCommentToCourse)\
.pipe(Trening.groupData)\
.pipe(Trening.calculateSpeed)\

Trening.dnfCountandReplace()

@staticmethod som leser inn .csv filen. Denne tar inn to nødvendige argumenter: path og fileName. F.eks. path = "/Users/cmagelssen/Desktop/DataAnalyse/data/pilot1/trening1/", filename = "PILOT1_SESSION2.csv". x = Trening.loadData(path, filename)

Trening.changeDataType()

@staticmethod som endrer datatype på hele datasettet

Trening.renameCommenttoCourse()

@staticmethod som endrer kolonnenavn fra COMMENT til COURSE. Den endrer samtidig

Trening.groupData()

@staticmethod som grupperer etter BIB# (dvs. utøver) og etter løype. Man bør velge ENTEN denne metoden eller groupDataAndFindTwoBestRunsForEachCourse(). Ikke begge to samtidig

Trening.groupData()

@staticmethod regner ut inngangshastighet

Modulen lageplot

Modulen 'lageplot' består av en klasse som heter 'Visualisering'. Denne innholder en rekke statiske metoder (@staticmethods) som lager plot av dataen. For at disse metodene skal fungere er det viktig at

visualisering.Prestasjon()

@staticmethod som lager et enkelt plot med loyper som X variabel og FINISH som Y variabel. COURSE er hue. Krever en dataframe som argument

visualisering.plotInngangshastigetForLoyper()

@staticmethod som lager et enkelt plot for inngangshastighet for de ulike løypene per utøver. Tar X, Y og HUE og krever en DataFrame som et argument
